## Приложение "Заметки"
Используется Spring Boot, Spring Data JPA, Lombok, H2 на диске
### Сборка и запуск 
```bash
  mvn spring-boot:run
```
### Использование
Открыть в браузере:
http://localhost:8081
### Функции
<ul>
<li>Просмотр всех заметок</li>
<li>Добавить заметку</li>
<li>Изменить заметку</li>
<li>Удалить заметку</li>
</ul>
